# minishell
### Aussi mignon qu'un vrai shell

### Projet 42
### L'objectif de ce projet est de créer un simple shell.

#### Enjeux et intérêts pédagogiques :
1. Parsing en plusieurs étapes :
  - Analyse syntaxique
  - Expansion des paramètres et variable d'environnement
  - Tokenisation
2. Utilisation des forks pour créer des processus enfants
2. Utilisation des pipes pour la communication entre les processus
